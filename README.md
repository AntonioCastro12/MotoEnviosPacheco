# Moto Envios Pacheco - Tarjeta digital

Tarjeta digital en React para compartir por link o abrir en celular.

Incluye:

- Foto y logo del servicio
- Botones de WhatsApp, llamada y SMS
- QR directo a WhatsApp
- Boton para guardar contacto `.vcf`
- Servicios: comida, paquetes, pagos y mandados

## Ejecutar

```bash
npm install
npm run dev
```

## Generar version final

```bash
npm run build
```

La version generada queda en `dist/`.
